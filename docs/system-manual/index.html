1. 系統概述

本系統為多公司集團企業資源規劃（ERP）系統，支援集團級別的跨公司管理。系統採用現代化的網頁技術開發，可在任何支援現代瀏覽器的裝置上使用。

1.1 系統架構

集團管理

支援多集團架構，每個集團可管理多家公司

公司管理

每家公司獨立的部門、職位、員工管理

跨公司調動

員工可在集團內不同公司間調動或兼任

彈性權限

基於角色、部門、職位的多層級權限控制

Google Chrome 90+ 推薦
Microsoft Edge 90+ 推薦
Mozilla Firefox 88+ 支援
Safari 14+ 支援

2. 登入與帳號

2.1 一般登入

使用您的員工帳號（員工編號或電子郵件）和密碼登入系統。

  1. 開啟系統網址，進入登入頁面
  2. 輸入您的帳號（員工編號或電子郵件）
  3. 輸入密碼
  4. 點擊「登入」按鈕
登入頁面截圖

2.2 QR Code 快速登入

使用已登入的行動裝置掃描 QR Code 快速登入。

  1. 在登入頁面點擊「QR Code 登入」
  2. 使用手機上已登入的 APP 掃描 QR Code
  3. 在手機上確認登入
  4. 網頁自動跳轉至儀表板

2.3 忘記密碼

  1. 點擊登入頁面的「忘記密碼」連結
  2. 輸入您的電子郵件地址
  3. 系統將寄送密碼重設連結至您的信箱
  4. 點擊郵件中的連結設定新密碼
密碼重設連結有效期限為 24 小時，請盡快完成重設。

4. 出勤管理

管理您的每日出勤打卡紀錄。

4.1 打卡功能

上班打卡

  1. 進入「出勤管理」頁面
  2. 確認目前時間顯示正確
  3. 點擊「上班打卡」按鈕
  4. 系統記錄打卡時間並顯示確認訊息

下班打卡

  1. 進入「出勤管理」頁面
  2. 點擊「下班打卡」按鈕
  3. 系統記錄下班時間並計算當日工時
出勤打卡介面截圖

4.2 出勤紀錄查詢

可查詢過去的出勤紀錄，包含：

  • 每日打卡時間（上班/下班）
  • 工作時數統計
  • 遲到/早退紀錄
  • 加班時數

4.3 班別設定

系統支援多種班別設定，由管理員設定後套用：

班別 上班時間 下班時間 彈性
正常班 09:00 18:00 ±30分鐘
早班 08:00 17:00 ±30分鐘
彈性班 08:00-10:00 17:00-19:00 核心時間制
彈性班需在核心時間（10:00-16:00）期間在崗。

5. 請假管理

申請、查詢和管理各類假別。

5.1 新增請假申請

  1. 進入「請假管理」頁面
  2. 點擊「新增請假」按鈕
  3. 選擇假別類型（特休、病假、事假等）
  4. 選擇請假日期與時間
  5. 填寫請假事由
  6. 如需要，上傳相關證明文件
  7. 點擊「送出申請」
請假申請表單截圖

5.2 假別說明

特別休假 依年資 年度特休假 否

事假 14天 私人事務 否

5.3 請假餘額查詢

在請假管理頁面上方可查看各假別的：

  • 年度總額度
  • 已使用天數
  • 剩餘天數
  • 預計請假（已申請尚未核准）

5.4 請假紀錄

可依以下條件篩選查詢：

  • 日期範圍
  • 假別類型
  • 審核狀態（待審核/已核准/已駁回）
已核准的請假如需取消，請聯繫人事部門處理。

6. 費用報銷

申請各類業務費用的報銷。

6.1 新增報銷申請

  1. 進入「費用報銷」頁面
  2. 點擊「新增報銷」按鈕
  3. 選擇報銷類型（交通、餐費、住宿等）
  4. 填寫費用明細（日期、金額、說明）
  5. 上傳發票或收據照片
  6. 確認總金額無誤
  7. 點擊「送出申請」
費用報銷表單截圖

6.2 報銷類別

交通費

計程車、高鐵、飛機、油資、停車費

餐費

業務用餐、加班餐費、客戶招待

住宿費

出差住宿、會議住宿

辦公用品

文具、耗材、小型設備

通訊費

業務電話費、網路費

其他

無法歸類的其他費用

6.3 注意事項

請確保上傳的發票/收據清晰可辨，包含日期、金額、商家名稱。電子發票請截圖完整交易明細。
  • 報銷金額需與發票金額一致
  • 超過 NT$3,000 需主管核准
  • 超過 NT$10,000 需部門主管核准
  • 申請後 30 天內需完成審核

7. 審核中心

查看和處理需要您審核的各類申請。

7.1 待審核清單

審核中心顯示所有需要您處理的申請，包括：

  • 請假申請
  • 費用報銷
  • 用印申請
  • 名片申請
  • 文具申請
審核中心清單截圖

7.2 審核操作

核准申請

  1. 點擊待審核項目查看詳情
  2. 確認申請內容無誤
  3. 可選擇填寫審核意見
  4. 點擊「核准」按鈕

駁回申請

  1. 點擊待審核項目查看詳情
  2. 填寫駁回原因（必填）
  3. 點擊「駁回」按鈕
  4. 申請人將收到駁回通知

退回重填

  1. 點擊待審核項目查看詳情
  2. 填寫需修改的說明
  3. 點擊「退回重填」按鈕
  4. 申請人可修改後重新送出

7.3 代理審核

當您因故無法處理審核時，可設定代理人：

  1. 進入「流程管理」→「代理人設定」
  2. 選擇代理人
  3. 設定代理期間
  4. 選擇可代理的申請類型
  5. 儲存設定
代理人需與您有相同或更高的審核權限。

8. 內部訊息

與公司同事進行即時通訊。

8.1 對話清單

左側顯示所有對話，包含：

  • 對方頭像與姓名
  • 最新訊息預覽
  • 未讀訊息數量（紅點）
  • 最後訊息時間
訊息清單截圖

8.2 發送訊息

  1. 點擊「新對話」按鈕
  2. 搜尋並選擇要傳訊的對象
  3. 在輸入框輸入訊息
  4. 按 Enter 或點擊送出按鈕發送

8.3 附加檔案

  1. 點擊輸入框旁的迴紋針圖示
  2. 選擇要傳送的檔案
  3. 確認檔案預覽
  4. 點擊送出

支援的檔案類型：

  • 圖片：JPG、PNG、GIF（最大 10MB）
  • 文件：PDF、Word、Excel（最大 10MB）
  • 文字：TXT（最大 1MB）

8.4 訊息狀態

圖示 狀態 說明
（無） 發送中 訊息正在傳送
✓ 已送達 訊息已存入伺服器
✓✓ 已讀取 對方已閱讀訊息
點擊 Header 的訊息圖示可快速查看未讀訊息。

9. 行政管理

處理各類行政申請，包含用印、名片、文具。

9.1 用印申請

申請公司印章用於正式文件。

  1. 進入「行政管理」→「用印申請」
  2. 點擊「新增申請」
  3. 選擇印章類型（可複選）
  4. 填寫用印事由與份數
  5. 上傳需用印的文件
  6. 送出申請

印章類型

印章 用途
公司章 一般公文、合約、證明文件
合約章 正式合約文件
發票章 發票、收據
銀行章 銀行相關文件
董事會章 董事會決議文件
騎縫章 多頁文件騎縫
用印申請表單截圖

9.2 名片申請

  1. 進入「行政管理」→「名片申請」
  2. 確認個人資訊（姓名、職稱、聯絡方式）
  3. 選擇名片樣式與數量
  4. 填寫特殊需求（如有）
  5. 送出申請

9.3 文具申請

  1. 進入「行政管理」→「文具申請」
  2. 瀏覽可申請的文具項目
  3. 選擇需要的品項與數量
  4. 填寫領取地點
  5. 送出申請

10. 人事管理

管理公司員工資料、組織架構、調動與離職。（需人事管理權限）

10.1 員工管理

查看員工清單

可依以下條件篩選：

  • 所屬公司
  • 部門
  • 職位
  • 在職狀態
  • 姓名或員工編號搜尋
員工清單截圖

新增員工

  1. 點擊「新增員工」按鈕
  2. 填寫基本資料（姓名、身分證、生日等）
  3. 填寫聯絡資訊（電話、地址、緊急聯絡人）
  4. 設定任職資訊（公司、部門、職位、到職日）
  5. 設定系統帳號（電子郵件、初始密碼）
  6. 儲存並通知員工

編輯員工資料

  1. 從員工清單點擊要編輯的員工
  2. 點擊「編輯」按鈕
  3. 修改需要更新的欄位
  4. 儲存變更
修改員工資料會自動記錄變更歷程，可在「變更紀錄」中查詢。

10.2 部門管理

管理公司的部門架構。

  • 新增/編輯/刪除部門
  • 設定部門層級（上級部門）
  • 指定部門主管
  • 查看部門人員清單

10.3 職位管理

管理公司的職位定義。

  • 新增/編輯/刪除職位
  • 設定職位等級（Level 1-10）
  • 設定職位類別（管理職/專業職/行政職）

10.4 員工調動

處理員工的部門或公司調動。

  1. 進入「人事管理」→「員工調動」
  2. 選擇要調動的員工
  3. 選擇新的公司/部門/職位
  4. 設定生效日期
  5. 填寫調動原因
  6. 送出調動申請

10.5 離職管理

處理員工離職流程。

  1. 進入「人事管理」→「離職管理」
  2. 選擇離職員工
  3. 填寫離職日期與原因
  4. 執行離職交接程序
  5. 完成離職作業

10.6 變更紀錄

查詢所有員工資料的變更歷程。

變更紀錄截圖

11. 組織圖

11.1 查看組織圖

  1. 進入「組織圖」頁面
  2. 選擇要查看的公司或集團
  3. 系統顯示階層式組織圖
  4. 點擊節點可查看詳細資訊
組織圖截圖

12.1 流程清單

系統支援以下類型的審核流程：

  • 請假流程
  • 費用報銷流程

12.2 流程編輯器

使用視覺化編輯器設計審核流程：

  1. 選擇要編輯的流程類型
  2. 進入流程編輯器
  3. 拖放節點建立流程
  4. 設定各節點的審核人員
  5. 設定條件分支（如金額、假別）

節點類型

開始 流程起點

會簽 多人同時審核
結束 流程終點

  1. 進入「流程管理」→「代理人設定」
  2. 點擊「新增代理」
  3. 選擇代理人
  4. 設定代理期間
  5. 選擇可代理的流程類型
  6. 儲存設定

12.4 員工審核路徑

查看每位員工的申請會經過哪些審核人員。

13. 財務會計

管理公司的財務會計作業。（需財務管理權限）

13.1 會計科目

管理會計科目表，包含：

  • 資產類科目
  • 負債類科目
  • 權益類科目
  • 收入類科目
  • 支出類科目
會計科目截圖

13.2 傳票管理

新增傳票

  1. 進入「財務會計」→「傳票管理」
  2. 點擊「新增傳票」
  3. 選擇傳票類型（收入/支出/轉帳）
  4. 填寫傳票日期與摘要
  5. 輸入借方與貸方科目及金額
  6. 確認借貸平衡
  7. 儲存傳票
傳票借方與貸方金額必須相等（借貸平衡）。

13.3 會計期間

管理會計期間的開啟與關閉：

  • 開啟新會計期間
  • 關閉會計期間（需主管核准）
  • 查看各期間傳票數量與金額

13.4 客戶/廠商管理

管理應收帳款客戶與應付帳款廠商：

  • 新增/編輯客戶資料
  • 新增/編輯廠商資料
  • 查看往來明細
  • 管理付款條件

13.5 財務報表

產出各類財務報表：

  • 日記帳
  • 總分類帳
  • 試算表
  • 資產負債表
  • 損益表

14.1 出勤報表

在通知中心可以：

  • 未讀通知數量（紅點）

18. 權限說明

系統採用三層式權限架構。

18.1 基本權限

所有員工登入後預設擁有的權限：

功能模組 權限
儀表板 完整
出勤管理 打卡、查看自己
內部訊息 完整
請假管理 申請、查看自己
費用報銷 申請、查看自己
審核中心 查看自己進度
行政申請 用印/名片/文具
組織圖 唯讀
人事管理 只能看自己
流程管理 隱藏
財務會計 隱藏
報表中心 隱藏
系統管理 隱藏

18.2 部門/職位加成權限

依據員工的部門或職位自動給予額外權限：

條件 額外權限
部門 = 財務會計部（非總務職位） +財務會計
職位 = 總務（一般） +人事管理（同公司）
職位 = 總務（主管級，Level ≥ 5） +人事管理（跨公司）

18.3 特殊權限

由管理員個別授予的特殊權限，具有最高優先級：

  • 可疊加在基本權限之上
  • 可覆蓋部門/職位權限
  • 可設定到期時間
  • 需記錄授權原因
特殊權限的優先級：特殊權限 > 部門/職位權限 > 基本權限

18.4 集團管理員

擁有 GROUP_ADMIN 權限的使用者可管理整個集團，包括：

  • 所有公司的人事管理
  • 所有流程的管理
  • 系統設定
  • 權限管理
